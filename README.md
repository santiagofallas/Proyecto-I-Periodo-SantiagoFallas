# Proyecto-I-Periodo-SantiagoFallas
Esta aplicación web sirve para administrar contactos, permitiendo guardar, ver, editar y borrar información fácilmente. Funciona muy bien en celulares y computadoras gracias a su diseño cómodo y moderno. Fue programada con Python y Flask para la lógica, mientras que la interfaz se armó con HTML y Bootstrap basándose en un diseño de Figma.
